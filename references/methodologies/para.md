# PARA

## 是什么
Tiago Forte 的信息组织法：按**行动性**把资料分四类——**P**rojects（有截止的项目）/ **A**reas（长期责任领域）/ **R**esources（将来可能有用）/ **A**rchives（完结·归档）。

## 怎么用（步骤）
1. 一条信息进来先问：**它服务于哪个正在做的项目**？→ Projects。
2. 不属于具体项目、但我长期要维护的领域（健康 / 财务 / 某技能）？→ Areas。
3. 都不是、只是"以后可能有用"？→ Resources。
4. 完结 / 不再活跃 → Archives。

## 什么时候用
资料多、乱、找不到时；想按"能不能用上"而不是"是什么主题"来组织。

## 一个例子
一篇"谈判技巧"：正在谈的那个合同 → Projects；泛泛提升沟通 → Areas；单纯觉得好 → Resources。

> 本第二大脑用「四层 × 六大工作模式」，PARA 作为**思路**参考——行动性可映射到 `30-Outputs`(项目产出) / `60-Domains`(领域) / `10-Inputs`(资料) / `90-Archive`。
