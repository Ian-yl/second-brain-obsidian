# 间隔重复

## 是什么
按**遗忘曲线**安排复习——刚学完密集复习，之后间隔越拉越长（1 天→3 天→1 周→1 月…），用最少的复习把知识钉进长期记忆。

## 怎么用（步骤）
1. 把要记的做成**问答卡**（一问一答，别整段背）。
2. 答对 → 下次间隔拉长；答错 → 间隔归零、重新密集。
3. 交给工具排期（Anki / 复习清单），到点就复习。
4. 只复习"快忘了"的，别浪费时间在已牢固的上。

## 什么时候用
背单词 / 术语 / 公式 / 考试 / 任何需要长期记住的事实性知识。

## 一个例子
背英语单词别一天塞 100 个然后忘光；做成卡片，Anki 按你的记忆强度排期，每天 10 分钟复习"快忘的那些"，长期留存率远高于突击。
