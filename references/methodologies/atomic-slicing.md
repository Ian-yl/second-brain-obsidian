# 切片法

## 是什么
又叫**原子笔记**。把大块知识**切成"一念一片"**——每篇只讲一个概念 / 结论，能独立看懂；再用 `[[链接]]` 把相关的连成网。

## 怎么用（步骤）
1. 遇到一个大主题，先**拆成若干独立小点**（每个小点能一句话说清）。
2. 每个小点写成**一篇短笔记**（标题=这个点、正文几句话，脱离上下文也读得懂）。
3. 用 `[[]]` 把相关笔记**连起来**（"这个让我想到那个"）。
4. 别追求一次写全——让知识网**随用随长**。

## 什么时候用
整理复杂主题、建可复用的知识网、避免笔记变成"一大坨堆料、以后找不到"。

## 一个例子
学"机器学习"别写一篇万字长文；切成：`[[什么是过拟合]]`、`[[训练集vs测试集]]`、`[[梯度下降一句话]]`… 每篇独立、互链，下次单独复用某一片很方便。
