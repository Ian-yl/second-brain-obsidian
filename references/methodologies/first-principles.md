# 第一性原理

## 是什么
把问题**拆到不能再拆的基本事实**（第一性原理），再从这些事实**从头往上推**——而不是靠类比 / 惯例 / "别人都这么做"。

## 怎么用（步骤）
1. 写下你对某问题的现有结论 + 它依赖的假设。
2. 逐条问「这真成立吗？凭什么？」——剥掉借来的假设。
3. 剥到**不可再质疑的基本事实**为止。
4. 只用这些事实**重新推导**，往往得到和惯例不同的答案。

## 什么时候用
遇到"大家都说不行" / 想突破惯例 / 做重大决策 / 拆解复杂问题时。

## 一个例子
马斯克造火箭：不问"火箭市价多少"（类比），而问"火箭由什么材料构成、这些材料多少钱"（第一性）→ 发现自造成本远低于采购价。
