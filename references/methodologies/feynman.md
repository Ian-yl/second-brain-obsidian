# 费曼学习法

## 是什么
通过「把概念用大白话教给一个小白」来学习——**讲不清楚的地方，就是你没真正懂的地方**。

## 怎么用（步骤）
1. 选一个你想搞懂的概念，写在纸 / 笔记顶端。
2. 假装教给一个完全不懂的人，**只用大白话、不用术语**讲一遍。
3. 卡壳、含糊、要靠术语糊弄的地方 → **标出来，那就是盲区**，回原始资料补。
4. 补完再讲一遍，精简到顺畅、连小孩都听懂为止。

## 什么时候用
学新概念、检验"我是不是真懂"、准备讲解 / 写作 / 面试前。

## 一个例子
学"复利"：别背公式，试着跟朋友说「钱生的利息，下一轮也跟着生利息，滚雪球」——如果你能顺畅打这个比方，你就懂了；如果只会念公式，说明还没懂。
