# 卡片盒笔记法

## 是什么
卢曼的笔记系统（Zettelkasten）：每条想法写成一张**永久笔记**（用自己的话、独立成篇），靠**链接**互相引用——笔记多了会自己"长出"新想法（知识复利）。

## 怎么用（步骤）
1. 闪念先记「临时笔记」（随手、不讲究）。
2. 定期把有价值的改写成「永久笔记」：**用自己的话、一念一篇、能独立看懂**。
3. 给每篇稳定标题，**链接到相关的已有笔记**（重点是链接，不是分类）。
4. 建「索引 / MOC」当入口——靠链接找，别靠文件夹深埋。

## 什么时候用
长期做研究 / 写作 / 积累领域知识；想让知识**复利式增殖**而不是堆积。

## 一个例子
读到"复利"→ 写一篇《复利一句话》→ 链到已有的《指数增长》《习惯的复利》——下次写文章，顺着链接就把相关想法都串起来。
